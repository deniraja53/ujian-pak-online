**Cara Melakukan Clone**
Untuk mendapatkan salinan kode proyek ini di komputer lokal Anda, ikuti langkah-langkah berikut:

1. Persiapan
Pastikan Anda sudah menginstal Git di perangkat Anda. Jika belum, unduh di git-scm.com.

2. Proses Cloning
Buka terminal (Command Prompt, Git Bash, atau Terminal di VS Code), lalu jalankan perintah berikut:
## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
